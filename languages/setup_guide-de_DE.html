Kurzanleitung zum Einrichten des Plugins

Erstmaliges Einrichten

Mehrere Einstellungen müssen konfiguriert werden, um die verschiedenen Funktionen dieses Plugins nutzen zu können. Diese Seite ist ein Tutorial zur Installation und Ausführung des Plugins auf deiner Website. Zuerst musst du eine allgemeine Vorstellung davon haben, wie du das Plugin verwenden willst, welche Informationen du sammeln und speichern willst und wie die Benutzer mit der Datenbank interagieren können.

Wie das Plugin eingerichtet werden kann

Angenommen, du möchtest ein Registrierungsformular haben, das die Namen und E-Mail-Adressen der Benutzer sammelt. Wenn Benutzer das Formular ausfüllen und absenden, werden sie auf eine andere Seite weitergeleitet, die ihnen dankt. Sie erhalten auch eine E-Mail, die die Registrierung bestätigt und ihnen mitteilt, wie sie die Informationen online ausfüllen können. Zu diesem Zweck wird ihnen in der E-Mail ein so genannter "Private Link" angegeben.

Darüber hinaus erhält der Administrator eine E-Mail-Benachrichtigung über die Registrierung mit einem direkten Link zum neuen Datensatz in der Datenbank.

Einrichten der Website

  1. Füge den Shortcode "[pdb_signup]" an der Stelle der Registrierungsseite ein, auf der du möchtest, dass das Registrierungsformular angezeigt wird.
  2. In den Einstellungen des Plugins wähle die Registerkarte "Registrierungsformular".
  3. Wähle die Bestätigungsseite aus, auf die der Benutzer weitergeleitet werden soll und füge den Shortcode "[pdb_signup_thanks]" auf dieser Seite ein.
  4. Wenn der eingeschriebene Benutzer auf den Link klickt, den er in der Bestätigungs-E-Mail bekommen hat (seinen "Private Link"), greift er auf die Datensatz-Formularseite zu, wo er durch das Datensatz-Formular seine Daten aktualisieren oder zusätzliche Informationen eingeben kann.
  5. In der Registerkarte "Datensatz-Formular" wähle die Seite aus, die dieses Formular anzeigen soll und füge auf dieser Seite den Shortcode "[pdb_record]" ein. Dieser Shortcode wird das Formular nur anzeigen, wenn es auf die Seite durch den "Private Link" zugegriffen wird.

Einrichten der Listenanzeigen und der Detailansicht

Du kannst auf deiner Website eine Liste aller eingeschriebenen Leute anzeigen. Jedes Element in dieser Liste kann einen Link enthalten, um detaillierte öffentliche Informationen über den entsprechenden Datensatz anzuzeigen. So richtest du das ein:

  1. Füge den Shortcode "[pdb_list]" auf der Seite ein, die die Mitgliederliste anzeigen soll.
  2. Auf der Seite "Verwalten von Listenspalten" im Admin-Menü des Plugins wähle die verschiedenen Felder aus, die du in der öffentlichen Liste zeigen möchtest, indem du sie aus dem Bereich "Verfügbare Felder" ziehst und in den Bereich "Listenspalten" ablegst.
  3. Auf der Detailansichtsseite, d.h. der Seite, wo die Detailansicht eines Datensatzes angezeigt werden soll, füge den Shortcode "[pdb_single]" ein. Dieser Shortcode wird alle Felder des Datensatzes anzeigen, deren Ansichtsmodus öffentlich ist, vorausgesetzt, dass der Zugriff auf die Detailansichtsseite mit der ID des betreffenden Datensatzes erfolgt, wie z.B. im: /participants/detail?pid=27
  4. In der Registerkarte "Listenansicht" der Plugin-Einstellungen wähle das mit der Detailansichtsseite zu verlinkende Feld aus.
  5. Zuletzt in der Detailansicht-Dropdown-Liste wähle die Detailansichtsseite aus, worin du den Shortcode "[pdb_single]" eingefügt hast.

Du kannst jetzt den Menüeintrag "Mitglied hinzufügen" wählen, dadurch ein erstes Mitglied hinzufügen, und dann ein zweites, und so weiter. Schließlich kannst du die gesamte Palette der angebotenen Funktionen überprüfen.
