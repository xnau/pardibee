Base de données des participants

Guide de configuration rapide

Configuration initiale

Il y a plusieurs éléments à configurer pour utiliser les différentes fonctionnalités de cette extension. Cette page est un tutoriel pour installer et faire tourner l'extension sur votre site. La première chose est d'avoir une idée générale de la façon dont vous voulez utiliser l'extension, quelle information vous souhaitez collecter et stocker, et comment les utilisateurs pourront interagir avec la base.

Exemple de configuration de l'extension

Supposons que vous voulez avoir un formulaire d'inscription qui recueille le nom et l'adresse mail des utilisateurs. Quand ceux-ci remplissent et soumettent le formulaire, ils sont redirigés vers une autre page qui les remercie. Un email leur est également envoyé pour confirmer leur inscription, et leur indiquer comment compléter les informations en ligne. Pour cela un lien privé leur est fourni dans l'email.

Par ailleurs, l'administrateur du site reçoit un email de notification de l'inscription, avec un lien direct vers la nouvelle entrée dans la base.

Configuration du site

  1. Placer le shortcode "[pdb_signup]" dans la page d'inscription, c'est-à-dire celle où vous voulez que le formulaire d'inscription apparaisse.
  2. Dans la configuration de l'extension, sélectionner l'onglet "Réglages de la page d'inscription".
  3. Indiquer la page de post-inscription vers laquelle l'utilisateur est redirigé, et placer le shortcode "[pdb_signup_thanks]" sur cette page.
  4. Quand l'utilisateur inscrit clique sur le lien fourni dans son email de confirmation, il/elle accède à la page du formulaire détaillé, une page où l'utilisateur peut compléter les informations le/la concernant via un formulaire détaillé.
  5. Sélectionner l'onglet "Réglages du formulaire détaillé", et précisez la page sur laquelle ce formulaire figurera. Placez le shortcode "[pdb_record]" sur cette page. Ce shortcode ne produira aucun résultat si l'accès à la page se fait sans le lien personnel contenu dans l'email de confirmation d'inscription.

Configuration de l'affichage des listes et de l'affichage détaillé d'enregistrements

Vous pouvez afficher sur votre site une liste des gens qui se sont inscrits. Chaque élément de la liste peut comporter un lien permettant d'afficher de façon détaillée l'information à vocation publique de l’enregistrement correspondant. Voici comment configurer cela:

  1. Placer le shortcode "[pdb_list]" dans la page où vous voulez générer la liste de participants.
  2. Dans "Gérer l'affichage des listes" (menu de l'extension) choisir les champs qui doivent apparaître dans la liste dite publique, en les faisant glisser depuis la zone "Champs disponibles" et en les déposant à l'endroit souhaité dans la zone "Colonnes dans la liste".
  3. Dans la page où vous voulez faire apparaître l'affichage détaillé d'un enregistrement, placer le shortcode "[pdb_single]". Celui-ci affichera tous les champs de cet enregistrement pour peu que leur visibilité soit définie comme totale et à condition que l'accès à la page se fasse avec l'ID de l'enregistrement concerné, comme par exemple dans: /participants/detail?pid=27
  4. Dans l'onglet "Réglages d'affichage de liste" de la configuration de l'extension, préciser (menu déroulant) le champ qui portera le lien vers l'affichage détaillé.
  5. Ensuite, indiquer la page d'affichage détaillé, c'est à dire celle contenant le shortcode "[pdb_single]".

Vous pouvez maintenant choisir "Ajouter un participant" dans le menu de l'extension, ajouter une entrée pour tester, et vous en servir pour passer en revue l'ensemble des fonctionnalités.
